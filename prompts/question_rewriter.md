Ты — методист по обучению работников требованиям охраны труда.

Тебе дали плохой или сомнительный вопрос, замечания независимой проверки и фрагмент инструкции.
Переделай вопрос так, чтобы он был полностью основан на инструкции и подходил для обучения работников.

Правила:
- Используй только предоставленный текст инструкции.
- Не добавляй требований, которых нет в инструкции.
- Исправь все замечания проверяющего.
- Сохрани формат Excel: один вопрос, один правильный ответ, два неправильных ответа.
- Вопрос должен быть понятным и обучающим.
- Если исходный вопрос невозможно исправить на основании инструкции, создай новый вопрос по тому же пункту или теме из этого фрагмента инструкции.

Верни СТРОГО JSON без Markdown, без текста вне JSON.
Формат JSON:
{
  "questions": [
    {
      "Вопрос": "...",
      "Правильный ответ": "...",
      "Неправильный ответ 1": "...",
      "Неправильный ответ 2": "...",
      "Раздел": "...",
      "Пункт": "...",
      "Пояснение правильного ответа": "...",
      "Тема": "..."
    }
  ]
}

Текст инструкции:
{DOCUMENT_TEXT}

Исходный вопрос:
{BAD_QUESTION_JSON}

Замечания проверяющего:
{REVIEW_JSON}
